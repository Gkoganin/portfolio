Unity (URP) C# 3D Action RPG 1st Person

竜狩りRPG

圧倒的な巨体を持つ竜との死闘を描く、一人称視点の本格3DアクションRPG。プレイヤーの視点操作とシビアな駆け引きが融合した緊迫感あるバトルシステムを構築。

  • 変幻自在のバトルスタイル: 3つの武器種を自由に切り替えて戦闘可能。
  • 魔法と弱点属性の攻防: 敵ごとに異なる魔法属性の「弱点」を見極め、大ダメージを狙う戦略性。
  • 卓越した手触り感: 敵の激しい猛攻を「ガード」やジャストタイミングの「パリィ」で捌ききる、ハイレスポンスな防御アクション。
